About This Workshop

Data visualization is one of the most critical skills for anyone working with data. In this hands-on workshop, you'll learn how to transform raw data into compelling visual stories using Tableau, one of the industry's leading visualization tools.

Whether you're a complete beginner or looking to enhance your existing skills, this workshop will provide you with practical techniques and best practices that you can apply immediately in your work.

Prerequisites: Basic familiarity with spreadsheets (Excel/Google Sheets). No prior Tableau experience required.

What You'll Learn

Tableau interface and basic navigation
Connecting to various data sources
Creating basic charts and graphs
Designing interactive dashboards
Best practices for visual design
Sharing and publishing visualizations
